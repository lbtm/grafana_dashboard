Dynamic dashboard for Host metrics from Telegraf: 
  - cpu
  - disk
  - disk.io
  - processes
  - swap memory usage
  - memory usage
  - system load and uptime
  - network interface usage
  - inode usage

Dependencies:

  - GRAFANA 3.1.0 :
  - GRAPH
  - INFLUXDB 1.0.0
  - SINGLESTAT

Data Sources:
  - InfluxDB
